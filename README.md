## 功能

支付宝最近出了VR红包，看了一下，发现很容易破解，搞个小软件玩玩

填补算法是，定位到黑条，用黑条相邻的两块图片区域来填补黑条，最后进行高斯模糊处理。

## 一些信息

以下数据根据小米5算的，有兴趣的可以对自己手机做适配

手机屏幕分辨率：1080*1920

图片大小：420*420，位置：（330,1071）

h是上下两个图片区域的高度，x是黑条个数；黑条高度8，图片高度7；

420 = h×2+8×x+7×(x-1)   

420 = 11×2+8×27+7×26
